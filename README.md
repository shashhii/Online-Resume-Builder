# Online-Resume-Builder

![](https://img.shields.io/badge/-java-blue) ![](https://img.shields.io/badge/-resume-builder-blue) ![](https://img.shields.io/badge/-jsp-blue) ![](https://img.shields.io/badge/-servlet-blue) ![](https://img.shields.io/badge/-web-app-blue) 

## About

A web app that lets users create and download professional resumes. I built this to solve the problem of students not having access to good resume tools.

## Tech Stack

Java, JSP, Servlet, HTML, CSS

## Getting Started

```bash
git clone https://github.com/shashhii/Online-Resume-Builder.git
cd Online-Resume-Builder
```

## Features

- Clean and intuitive UI
- Fully functional backend
- Responsive design
- Well-structured codebase
